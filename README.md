## 📌 Project Overview

The **Loan Approval Prediction System** is a Machine Learning powered web application built using **Django** that predicts whether a loan application should be **Approved or Rejected**.

- The system analyzes applicant details like income, credit history, loan amount, and property area to make predictions using a trained ML model.

---

## 🎯 Problem Statement

- Banks receive thousands of loan applications. Manually verifying each application takes time.

- This system helps automate the **loan approval process using Machine Learning** to quickly determine whether a loan should be approved.

## ⚙️ Installation

### 1️⃣ Clone the repository

- git clone https://github.com/yourusername/Loan-Approval-Prediction-System.git


### 2️⃣ Go to project folder

- cd Loan-Approval-Prediction-System


### 3️⃣ Install dependencies

- pip install -r requirements.txt


### 4️⃣ Run the server

- python manage.py runserver
